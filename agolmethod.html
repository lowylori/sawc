ArcGIS Online Methodology

Log into ArcGIS Solutions, through the ArcGIS Online portal. Search for the relevant solution you are looking for; in this case it was “Parks and Grounds Management”. Click the “Deploy now” button to deploy this solution so your account.

Once deployed on your account, you are able to start editing the components you would like to use. Firstly, open both surveys in Survey123 Connect to be able to edit them. While doing this, make sure to update the titles and names on the surveys to match the current project. Once any edits are made to the survey to meet the requirements you are looking to meet, republish the survey. This can be done by clicking the publish button, on the edit panel.

Next, edit the Workforce project to add any relevant staff to the project and assign them roles. To add workers, go to your relevant workforce project and click the gear wheel called “configure”. Once in this window, you can add more mobile workers or dispatchers through “users” page on the top bar. Select the staff you would like added and a project role.

Next, open the surveys you created and add requests into them. This will be important for assigning requests to people. You can add requests through both the external and internal surveys to ensure they work.

If you logged out of Workforce, log back into Workforce and practice assigning an open task to a staff member. This is to test out that the task assigns properly and that this change is reflected in the dashboards explored in step 6. To do this, Open the Workforce project by clicking it. You can either select a request on the map by clicking on it and click “create assignment” to bring up a window to add assignment type and relevant details such as priority, worker assigned to as well as add any more details to the request. To push it through, click “create assignment”. New assignments can also be created by managers by clicking the “+ Assignments” button on the top bar. The same as above, fill in the details and assign it to a worker. Now workers will be able to see tasks assigned to them.

Next, edit the dashboards and requests centers to match clients name and needs. By selecting and editing each dashboard separately from your contents page, update the titles and names of the dashboards to match the project. You can do this by clicking “edit dashboard” and once in the dashboard, you can edit every part of the dashboard by clicking the “…” over every component and opening the settings, called “configure”. On the dashboard, ensure the data being presented makes sense in relation to the project goals and select new questions to be presented in the graph area if needed.

Utilize ArcGIS API for Python within ArcGIS Online Jupyter Notebook environment to do the following:

  • Create a New Jupyter Notebook
  • Access and Load Survey123 Data
  • Create Reporting Templates
  • Create Tasks
  • Automate Reporting
  • Set up task scheduling

Connecting Jupyter Notebooks to ArcGIS Online Survey123 makes it easy to select data for reports. By automating the notebook with tasks, reports can be generated regularly, whether daily, weekly, or monthly. This setup simplifies the process and ensures timely updates for customized reports. Although we're currently unable to automate report delivery to the client, we plan to gather feedback and work towards implementing their preferred method whenever we are on-site.
